🐾 Mindful Paws – Pet Grooming Landing Page
📌 Project Overview

Mindful Paws is a simple, responsive landing page for a boutique pet grooming salon.
It is built using pure HTML (no frameworks or external dependencies) and showcases a clean UI with interactive elements.

✨ Features
🏠 Hero Section with business name and calming tagline
🧠 Interactive Booking Quiz
Select pet type (Dog/Cat/Other)
Choose size (Small/Medium/Large)
Pick a service
Get instant price estimate
💼 Services Section
Bath & Brush ($45)
Full Groom ($75)
Spa Day ($120)
Puppy First Groom ($55)
🖼️ Gallery Section with pet image placeholders
📅 Booking Form
Pet Name
Owner Name
Phone Number
Preferred Date
🛠️ Technologies Used
HTML5
CSS3 (optional if styled)
JavaScript (for quiz interactivity)
